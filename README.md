## 2. Factorial

### Description
Determine the <a href="https://en.wikipedia.org/wiki/Factorial">factorial</a> of **n**-th number.

### The functionality of the program
* Calculating factorial using:
  * recursions
  * cycle
* Comparison of working time

### Example
```
Enter n (NUMBER): 
5
            Recursion    Cycle     
Result      120          120       
Time, ns    3300         1100      
```
